# 🎬 NullOfflineVideoTranslator (EN ➡️ RU)

Полностью автономный пайплайн для перевода англоязычных видео на русский язык. Программа сама скачивает видео, вырезает музыку, распознает текст, переводит его с учетом сленга (через ИИ) и озвучивает клонированным голосом, автоматически подгоняя звук под видеоряд.

## ✨ Ключевые фишки (Что внутри?)

- ⬇️ **Smart Download (`yt-dlp`):** Автоматическое скачивание видео с YouTube в максимальном качестве 1080p. Названия файлов сами очищаются от спецсимволов.
- 🎵 **Studio-Grade Separation (`Demucs`):** Изоляция голоса. Нейросеть физически отделяет фоновую музыку, басы и эффекты от человеческой речи, чтобы транскрибация была идеальной.
- 📝 **Forced Alignment (`WhisperX`):** Сверхбыстрое распознавание текста (CTranslate2). В отличие от обычного Whisper, привязывает каждое слово к миллисекундам на таймлайне.
- 🧠 **Context-Aware Translation (`Gemini 3.1`):** LLM-перевод целыми батчами. Идеально переводит геймерский сленг, мемы и нецензурную лексику без потери контекста. Поддержка балансировки ключей!
- 🗣️ **Zero-Shot Voice Cloning (`XTTSv2 INT8`):** Клонирование голоса оригинального диктора. Оптимизировано под CPU (INT8). Встроен алгоритм *умной нарезки* (если текст больше 150 символов) и *авто-ускорения* `ffmpeg atempo` для липсинка.
- 🎚️ **Dynamic Audio Leveling (Audio Ducking):** Умный микшер `pydub`. Автоматически анализирует громкость по шкале dBFS. Приглушает музыку до `-14 dBFS` и выводит русский голос на `-8 dBFS`, чтобы он всегда звучал отчетливо поверх любых взрывов и басов.
- ☁️ **Google Colab Playbook:** Встроенный готовый блокнот для запуска на серверных видеокартах Tesla T4 за пару минут.

## 🚀 Как запустить (Google Colab - Рекомендуется!)

Если вы не хотите ждать часы на слабом процессоре, запустите скрипт на мощных бесплатных серверах Google:
1. Откройте файл `Video_Translator.ipynb` в Google Colab.
2. В верхнем меню выберите `Среда выполнения` -> `Сменить среду выполнения` -> **T4 GPU**.
3. Заполните красивые интерактивные поля (Токен GitHub, ключи Gemini и ссылку на YouTube).
4. Нажмите кнопку **Play (▶️)**.
5. Заварите кофе ☕. Через 5 минут в панели слева появится готовый переведенный `.mp4` файл!

## 💻 Как запустить локально (На своем ПК)

### Шаг 1: Установка
Убедитесь, что у вас установлен Python (скрипт поддерживает даже Python 3.14!) и `ffmpeg`.
Запустите умный скрипт установки, который сам создаст виртуальное окружение и пропатчит все конфликты библиотек:
```bash
python3 install.py
```

### Шаг 2: Настройка секретов
В корне проекта скопируйте `.env.example` в `.env` и впишите свои ключи:
```ini
HF_TOKEN=hf_ВАШ_ТОКЕН_ОТ_HUGGINGFACE
GEMINI_API_KEYS=AIzaSy_ВАШ_КЛЮЧ_1, AIzaSy_ВАШ_КЛЮЧ_2
```
*(Скрипт автоматически переключается между ключами Gemini, если один из них словит лимит!)*

### Шаг 3: Погнали!
Просто скормите скрипту любую ссылку на YouTube:
```bash
./run.sh "https://www.youtube.com/watch?v=dQw4w9WgXcQ"
```
Скрипт умный: если он прервется, при следующем запуске он не будет скачивать видео заново или переводить уже готовые куски. Он просто продолжит с того места, где остановился!

## 🏗️ Архитектура (Под капотом)
1. **`run.sh`** -> Санитизация названия и скачивание видео.
2. **`utils.py`** -> Извлечение сырого аудио из `.mp4`.
3. **`audio_processor.py`** -> Работает Demucs. Разделяет аудио на `vocals.wav` и `no_vocals.wav`. В конце динамически сводит звук по уровню децибел (dBFS).
4. **`transcriber.py`** -> Загружает WhisperX, ищет голос (VAD) и достает английский текст с таймингами.
5. **`translator.py`** -> Формирует JSON из сегментов и отправляет пачкой в Google Gemini API.
6. **`tts_engine.py`** -> Клонирует голос через XTTSv2, режет длинный русский текст на предложения, генерирует звук и ускоряет его, чтобы попасть в тайминги оригинала.
7. **`main.py`** -> Дирижер, который управляет всем процессором и склеивает финальное видео.
